Comment participer / contribuer / aider l'espace coworking ?

Comment publier une actualité ?

Vous pouvez publier une actualité directement via le forum en plaçant votre message dans la catégorie actualités.

Où sont les statuts de l'association ?

Ils sont disponibles ici : « Statuts de l'association Coworking Metz de février 2014 ».

Comment envoyer un mail à tous les membres de Coworking Metz

Dans le compte Mailchimp de Coworking Metz, il y a un groupe nommé "Adhérent" qui contient l'adresse mail de tous les adhérents.

Est-ce qu'il existe une lettre d'actualités ? (newletter)

Oui, vous pouvez vous inscrire à la newletter de Coworking Metz à l'adresse suivante : http://coworking.a-metz.info/newletter.html.

Où est hébergé le site web ?

Le site web est hébergé sur un serveur dédié de Stéphane Klein

L'hébergement est très simple, 99% basé sur de simple fichiers statiques et deux pages php pour le paiement paypal.

Comment est développé le site web ?

Le site web est un site statique, le code source est librement accessible sur github.

Vous pouvez librement forker le projet et envoyer des pull request pour appliquer des modifications en ligne.

Vous pouvez même directement éditer les fichiers en ligne via github, exemple, si vous êtes connecté à votre compte github, vous pouvez éditer la page contribuer.html.
