Comment participer / contribuer / aider l'espace coworking ?

Comment publier une actualité ?

Vous pouvez publier une actualité directement via le forum en plaçant votre message dans la catégorie actualités.

Où sont les status de l'association ?

Ils sont disponibles ici : « Status de l'association Coworking Metz de février 2014 ».

Comment envoyer un mail à tous les membres de Coworking Metz

Dans le compte Mailchimp de Coworking Metz, il y a un groupe nommé "Adhérent" qui contient l'adresse mail de tous les adhérents.

Liste des services utilisés par l'association

Nom du service Rôle Comment y avoir accès
Mailchimp Envoi de newletter Demander à contact@stephane-klein.info
Google Drive Rassemble tous les documents administratifs, factures, lettres, listing, archives… Demander à contact@stephane-klein.info
